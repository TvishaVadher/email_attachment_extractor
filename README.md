# Smart Email Attachment Extractor
This project is a Python script that connects to a Gmail inbox and automatically extracts email attachments.  
The extracted files are organized into folders based on date, sender, and file type for easy access.

## Features
- Connects to Gmail using IMAP
- Automatically extracts email attachments
- Organizes files by date, sender, and file type

## Requirements
- Python 3.x
- Gmail account
- Gmail App Password
- IMAP enabled in Gmail settings

## How to Run
1. Enable IMAP in your Gmail settings  
2. Generate a Gmail App Password  
3. Add your Gmail email address and App Password in the script  
4. Run the script:
   ```bash
   python attachment_extractor.py
